name: Define Service Level Agreements (SLAs)

objective: Define the level of service standards provided to the customer during Operations and Maintenance (O&M).
responsibleParty: shared
category: process

activities: 
  - text: Review Target State Concept of Operations and roles and responsibilities for O&M services
    responsibleParty: shared
  - text: Review and discuss existing provider service level metrics to determine if there need to be adjustments. All service level metrics must align, and not conflict with, the metrics included in the <a href="https://www.ussm.gsa.gov/fibf/">Federal Integrated Business Framework (FIBF)</a> Performance Measures
    responsibleParty: shared
  - text: Identify additional service level metrics based on objective, clearly-defined, and measurable criteria, in alignment with the <a href="https://www.ussm.gsa.gov/fibf/">Federal Integrated Business Framework (FIBF)</a> Performance Measures
    responsibleParty: shared
  - text: Gain stakeholder buy-in on metrics that will be used post-migration to measure O&M performance as part of the performance management strategy
    responsibleParty: shared
  - text: Define performance management capabilities, processes, and tools
    responsibleParty: shared
  - text: Finalize service level metrics, targets, and remediation or issue resolution policies, in alignment with the <a href="https://www.ussm.gsa.gov/fibf/">Federal Integrated Business Framework (FIBF)</a> Performance Measures
    responsibleParty: shared
  - text: Report updates in governance meetings, and Status Reports/Dashboards, informing QSMOs as necessary
    responsibleParty: shared
